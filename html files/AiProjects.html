The image captioner

An application made using cv2, YOLO, Blip, and Tkinter with a team of 4 others. The application allows users to enter a video or image, which will then be identified. I created the front end for this application, and assisted in choosing the AI models.

Surveillance research project

A project created with 2 people investigates how law enforcement agencies across the U.S use surveillance technologies, as well as what it means for public privacy. Using our four datasets, we were able to create graphs and visualizations. We used visualization and SQL to find trends and patterns in our datasets. Using Random Forests for our classification model, we calculated the probability that each Technology would be used by a type of Law Enforcement Agency. We used F1 scores to check the precision and recall of our model.
